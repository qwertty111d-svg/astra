# Что делать дальше, очень просто

## 1. Этот архив уже содержит нужный код для приложения
В нём уже есть:
- /api/desktop/login
- /api/desktop/session
- /api/desktop/license
- /api/desktop/logout
- /api/desktop/device/register
- таблица desktop_devices

## 2. Что делать руками

### Шаг 1. Залей этот сайт на сервер вместо старого кода
Просто замени файлы сайта на сервере содержимым этого архива.

### Шаг 2. На сервере открой файл .env
Если файла .env нет, возьми .env.production.example, сделай копию и назови её .env.

### Шаг 3. В базе сайта выполни только один файл
Нужен только этот файл:

drizzle/0003_desktop_devices.sql

Если у тебя сайт уже работал раньше, файлы 0000, 0001, 0002 повторно НЕ трогай.

### Шаг 4. Перезапусти сайт
После загрузки файлов и SQL просто перезапусти сайт на сервере.

### Шаг 5. Проверь
Открой в браузере:
https://astraboost.ru/api/desktop/session

Если видишь JSON, а не 404, значит сайт уже умеет работать с приложением.

## 3. Что делать в приложении
В приложении открой файл:
apps/desktop/.env

И вставь туда:
API_BASE_URL=https://astraboost.ru
VITE_PRICING_URL=https://astraboost.ru/pricing

## 4. Всё
Теперь:
- сайт управляет входом и тарифами
- приложение только оптимизирует ПК
- кнопка Купить ведёт на сайт
